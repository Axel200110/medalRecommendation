# AESTHETIC: Talent Recommendation & Performance Evaluation System

A research-grade Multi-Criteria Decision Making (MCDM) framework and Performative Analytics Platform tailored for Sri Lankan cultural and aesthetic show competitions (such as Kandyan Dance, Low-country Dance, Geta Bera, and Baila).

## 🚀 Key Features

*   **Integrated Multi-Rater Consensus:** Real-time Cronbach's Alpha ($\alpha$) computation to calculate judge reliability.
*   **Time-Series Proficiency Degradation:** Built-in exponential skill decay models using Brain.js neural networks to forecast skill retention.
*   **NLP Resume Intake & Parser:** Automatic extraction of performer capabilities and local cultural skills from uploaded text resumes.
*   **Multimedia Showcase Integration:** Responsive inline players for YouTube performance clips, Google Drive videos, and HTML5 video files.
*   **Bi-Theme Interface:** Premium Arctic Pearl & Midnight Violet responsive layout built with glassmorphism aesthetics.
*   **SPSS & R Export Engine:** Direct CSV matrix download for downstream statistical model analysis.
*   **Supabase Sync:** Operational cloud database integration for participant storage.

## 🛠️ Tech Stack

*   **Core:** HTML5, CSS3, Vanilla JS
*   **Libraries:** Chart.js, Brain.js, Lucide Icons
*   **Database:** Supabase Cloud

## 📦 Local Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Axel200110/AESTHETIC.git
    cd AESTHETIC
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run development server:**
    ```bash
    npm run dev
    ```
